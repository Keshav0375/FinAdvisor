# 2.4 — Ingest Pipeline Script

## Problem
Need an end-to-end pipeline that takes the 50 synthetic corpus documents, redacts PII, chunks them for embedding, generates vector embeddings via Voyage AI, and loads everything into PostgreSQL (documents + chunks tables) ready for RAG retrieval.

## Implementation
- `backend/scripts/ingest.py` — Full ingest pipeline:
  - `load_corpus_docs()` — Reads all JSON files from `data/corpus/` recursively
  - `chunk_text()` / `_recursive_split()` — Recursive character text splitter (2048 chars, 200 overlap) with separators: `\n\n`, `\n`, `. `, ` `
  - PII redaction via `RegexRedactor` (default) or `PIIRedactor` (GCP DLP mode)
  - Batch embedding via `VoyageEmbeddings` (voyage-3, 1024 dims)
  - Raw SQL INSERT into `documents` and `chunks` tables
  - `--dry-run` flag: parse/chunk/redact only, no API calls or DB writes
  - `--target test` flag: uses `finadvisor_test` database for CI
  - Cost estimation logged before embedding begins

## Decisions
- Used 2048-char chunks (not 512-token) — simpler to implement without a tokenizer dependency, and roughly equivalent (~512 tokens ≈ 2048 chars). Produces 114 chunks from 50 docs (avg 2.3 per doc).
- Raw SQL INSERT instead of ORM — avoids session complexity with vector columns and keeps the script independent from potential ORM quirks with pgvector.
- Overlap uses character-level prepending from previous chunk tail — ensures context continuity at chunk boundaries.
- PII redaction happens BEFORE chunking — ensures no PII leaks into the vector store (CLAUDE.md non-negotiable rule #2).
- `datetime.now(UTC)` instead of deprecated `datetime.utcnow()`.

## Verification
- `python scripts/ingest.py --dry-run` — successful:
  - 50 docs loaded from corpus
  - PII detected in 15+ documents (regex fallback caught SSN, phone, email, account patterns)
  - 114 chunks produced (avg 2.3 per doc)
  - No API calls, no DB writes
- Full DB verification (INSERT + SELECT count) deferred until Docker/Postgres is available
- Quality gate: ruff, mypy, pytest all pass (10 passed, 7 skipped)
